# 🐍 Snake Game (Pygame)

A simple 2D Snake game built using Python and Pygame.  
This project demonstrates basic game development concepts such as movement, collision detection, and user input handling.

## 🎮 Features
- Snake movement using keyboard arrows
- Random food generation
- Score system
- Collision detection (walls and self)
- Game Over screen with restart option

## 🛠️ Technologies
- Python
- Pygame

## ▶️ How to Run

1. Install dependencies:

bash

pip install pygame
Run the game:
python snake.py

🎮 Controls

Arrow Keys → Move the snake
R → Restart the game after Game Over

📚 Learning Goals

Understand game loops
Handle keyboard input
Work with 2D graphics
Implement basic game logic

🚀 Future Improvements

Save high scores

# 🐍 Jogo da Cobrinha (Pygame)

Um jogo simples 2D da cobrinha desenvolvido com Python e Pygame.  
Este projeto demonstra conceitos básicos de desenvolvimento de jogos, como movimento, detecção de colisão e interação com o usuário.

## 🎮 Funcionalidades

- Movimento da cobra com as setas do teclado
- Geração aleatória de comida
- Sistema de pontuação
- Detecção de colisão (paredes e corpo)
- Tela de Game Over com opção de reiniciar

## 🛠️ Tecnologias
- Python
- Pygame

## ▶️ Como Executar

1. Instale as dependências:

bash
pip install pygame
Execute o jogo:
python snake.py

🎮 Controles

Setas → Movimentar a cobra
R → Reiniciar após perder

🚀 Melhorias Futuras
