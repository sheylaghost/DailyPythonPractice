# 🎓 Mini School Management System | Sistema Escolar Simples

## 🇧🇷 Português

### 📖 Sobre o Projeto

Este projeto é um mini sistema de gerenciamento escolar desenvolvido em Python utilizando arquivos CSV para armazenar dados.

O objetivo é praticar conceitos fundamentais de programação, como:

* Manipulação de arquivos CSV
* Estruturas de repetição
* Estruturas condicionais
* Funções
* Modularização
* CRUD (Create, Read, Update, Delete)
* Organização de projetos

### 🚀 Funcionalidades

* Cadastro de alunos
* Listagem de alunos
* Busca de alunos
* Atualização de informações
* Remoção de alunos
* Registro de notas
* Relatórios simples

### 🛠️ Tecnologias Utilizadas

* Python 3
* Biblioteca CSV

### 📂 Estrutura do Projeto

```text
escola/
│
├── main.py
├── alunos.csv
└── README.md
```

### 💡 Exemplo de Registro

```csv
nome,nota
Eyshila,10
Ana,8
Carlos,9
```

### 🎯 Objetivo de Aprendizagem

Este projeto foi criado para desenvolver habilidades em Python e compreender como sistemas de gerenciamento funcionam antes da utilização de bancos de dados relacionais.

## 🇺🇸 English

### 📖 About the Project

This project is a mini School Management System developed in Python using CSV files for data storage.

The goal is to practice fundamental programming concepts such as:

* CSV file handling
* Loops
* Conditional statements
* Functions
* Code organization
* CRUD operations (Create, Read, Update, Delete)

### 🚀 Features

* Student registration
* Student listing
* Student search
* Information update
* Student removal
* Grade registration
* Basic reports

### 🛠️ Technologies Used

* CSV Library
