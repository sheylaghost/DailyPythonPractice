Salary Calculator / Calculadora de Salário

📝 Descrição / Description

PT-BR: Script simples para calcular salário bruto, descontos (IR, INSS, Sindicato) e salário líquido.

EN: Simple script to calculate gross salary, taxes (Income Tax, Social Security, Union), and net salary.

🧮 Regras de Cálculo / Calculation Rules

IR: 11%

INSS: 8%

Sindicato / Union: 5%

🚀 Como usar / How to use

Certifique-se de ter o Python instalado.

Rode o comando no terminal:

Bash

python Horas.py

Insira o valor da sua hora e a quantidade de horas trabalhadas.

💻 Exemplo de Saída / Output
Plaintext

+ Salário Bruto : R$ 1000.00
- IR (11%) : R$ 110.00
- INSS (8%) : R$ 80.00
- Sindicato (5%) : R$ 50.00
= Salário Líquido : R$ 760.00
