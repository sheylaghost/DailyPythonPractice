Salary Calculator / Calculadora de Salário📄 

Description / Descrição

EN:

 This is a Python script designed to calculate a monthly salary report based on hourly wages and hours worked. It automatically calculates deductions for Income Tax (11%), Social Security (INSS - 8%), and Union dues (5%) to provide the final net salary.

 PT-BR:

 Este é um script em Python desenvolvido para calcular o relatório salarial mensal com base no valor ganho por hora e horas trabalhadas. Ele calcula automaticamente os descontos de Imposto de Renda (11%), INSS (8%) e Sindicato (5%) para fornecer o salário líquido final.

🧮 Calculations / Calculos

Item,Rate / Taxa
Gross Salary / Salário Bruto,Hourly rate × Hours / Valor hora × Horas
Income Tax / IR,11%
INSS,8%
Union / Sindicato,5%
Net Salary / Salário Líquido,Gross - (IR + INSS + Union)

🚀 How to Run / Como Executar
Prerequisites / Pré-requisitos
Python 3.x installed. / Python 3.x instalado.

Instructions / Instruções
Clone the repository / Clone o repositório:

Bash
git clone https://github.com/your-username/salary-calculator.git
Navigate to the folder / Entre na pasta:

Bash
cd salary-calculator
Run the script / Execute o script:

Bash
python Horas.py
